File to format:











































This tool performs some quick formatting on Dreamweaver documents pasted from word. It is NOT for general use since it takes shortcuts assuming Dreamweaver formatting. It also assumes proper indentation (in Dreamweaver, Edit -> Code -> Apply Source Formatting).

Be sure to apply source formatting with Dreamweaver afterwards to fix the indentation and so on.

Check the console (ctrl+shift+i) for any errors if nothing is loading.

More in-depth information can be found in the readme.
