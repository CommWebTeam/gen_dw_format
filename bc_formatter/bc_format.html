File to format:







This tool performs some quick formatting on WET-formatted HTML documents to remove false flags for mismatches in Beyond Compare.

Do NOT edit or otherwise use this tool's output as the actual document, since it is intentionally introducing formatting issues to make Beyond Comparing easier. Edit the original document instead.

Check the console (ctrl+shift+i) for any errors if nothing is loading.

More in-depth information can be found in the readme.
