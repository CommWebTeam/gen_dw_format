This tool converts WET 3 classes to WET 4 classes. Before using the tool, make sure that the HTML document:

  1. has proper indentation (in Dreamweaver, Edit -> Code -> Apply Source Formatting).
  2. uses double quotes (e.g. "x") instead of single quotes (e.g. 'x') for tag attributes.

Note that this tool only performs one-to-one replacements; you should do circumstantial replacements manually.

More in-depth information can be found in the readme.
